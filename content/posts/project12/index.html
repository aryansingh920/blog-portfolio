Fine-Tuning GPT-2 for Financial News and Analysis Text Generation

A Comprehensive Guide to Building a Financial News Generator Using GPT-2

Introduction

This project focuses on fine-tuning GPT-2, an advanced language model, to generate coherent and contextually relevant financial news articles. By leveraging the power of transfer learning, we create a system that can analyze and produce financial news headlines and summaries tailored to specific stock tickers or general market updates.

GitHub Repository: Fine-Tuning GPT-2

Technical Workflow

1. Data Preparation

The project begins by collecting financial news data using the getData.py script. It fetches news articles from Yahoo Finance for a list of stock tickers, extracting key details such as title, summary, description, publication date, and provider.

  • Data is read from a CSV file containing stock ticker symbols.
  • News articles are fetched using the Yahoo Finance API.
  • Processed data is stored in a CSV file for further use in model fine-tuning.

2. Fine-Tuning GPT-2

The prepare_fineTuning.py script fine-tunes GPT-2 on the prepared dataset. The process involves the following steps:

  • Converting the CSV data into a text format compatible with GPT-2 training.
  • Using the Hugging Face Trainer API to fine-tune the model.
  • Saving the fine-tuned model and tokenizer for later use.

3. Generating Financial News

The main.py script implements the FinancialGPT2Generator class. It loads the fine-tuned model and generates news articles based on user input. Users can specify:

  • A specific stock ticker to generate news for.
  • Text generation parameters such as maximum length and temperature.

4. Streamlit Application

The app.py script provides a user-friendly interface for interacting with the model. Built with Streamlit, the app allows users to:

  • Select random news generation or input specific stock tickers.
  • Adjust generation parameters dynamically.
  • Generate financial news headlines and summaries with a single click.

Results and Analysis

The fine-tuned model performs exceptionally well in generating coherent and contextually accurate financial news. Key observations include:

  • High relevance to specific stock tickers or general market conditions.
  • Flexible text generation, controlled by parameters like temperature and length.
  • Smooth real-time interaction through the Streamlit application.

How to Run the Project

  1. Clone the repository from GitHub.
  2. Prepare the dataset by running getData.py.
  3. Fine-tune the model using prepare_fineTuning.py.
  4. Run the Streamlit app with streamlit run app.py.
